Snipit of Gnuplot code so I don't have to reach through that documentation again :)

The script was originally created for modeling state dyanmics on the Bloch Sphere, but has been altered for more general usecases.
by default, this is designed for a datafile containing 4 or more columns, being:
  1) Datapoint number (Can be substituted by row number, but this was easier for my purposes)
  2) x data
  3) y data
  4) z data
  >4) Whatever else

Enjoy!
